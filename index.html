1
On your iPhone, press and hold Volume Up + Volume Down + Side Button simultaneously for 1–2 seconds, then release all three.
2
You’ll feel a short vibration. A sysdiagnose is now generating in the background. Wait 5–10 minutes — it takes time.
3
Go to Settings → Privacy & Security → Analytics & Improvements → Analytics Data.
4
Scroll down to find a file starting with sysdiagnose_ followed by a date and timestamp. It will be a .tar.gz file.
5
Tap it, then tap the Share icon. Choose Save to Files (or AirDrop to your computer).
6
Come back here and tap the box above to select the file. Then tap ANALYZE.
The sysdiagnose file is typically 200–300 MB. On iPhone, analysis uses more memory than on desktop. If analysis fails, try closing all other apps and tabs first, or AirDrop the file to a Mac/PC and run it there.
All analysis runs locally in your browser using JavaScript. No data is uploaded anywhere.
